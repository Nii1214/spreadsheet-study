# マークダウン記法チートシート

このプロジェクトで使用できるマークダウン記法の一覧です。

## 見出し

```markdown
# 見出し1（使用不可 - セクションタイトルとして使用）
## 見出し2
### 見出し3
#### 見出し4
```

**表示例：**
## 見出し2
### 見出し3
#### 見出し4

---

## 段落

通常のテキストはそのまま段落として表示されます。空行で段落を区切ります。

```markdown
これは最初の段落です。

これは2番目の段落です。
```

---

## 強調・太字

```markdown
**太字**
*斜体*（使用可能ですが、スタイリングは太字と同じ）
```

**表示例：**
**太字**

---

## リスト

### 順序なしリスト（箇条書き）

```markdown
- 項目1
- 項目2
  - ネストされた項目1
  - ネストされた項目2
- 項目3
```

**表示例：**
- 項目1
- 項目2
  - ネストされた項目1
  - ネストされた項目2
- 項目3

### 順序付きリスト（番号付き）

```markdown
1. 最初の項目
2. 2番目の項目
3. 3番目の項目
```

**表示例：**
1. 最初の項目
2. 2番目の項目
3. 3番目の項目

## コード

### インラインコード

```markdown
`コード` を文中に埋め込む
```

**表示例：**
`コード` を文中に埋め込む

### コードブロック

言語を指定してシンタックスハイライトを有効にできます。

````markdown
```javascript
function hello() {
  console.log("Hello, World!");
}
```

```python
def hello():
    print("Hello, World!")
```

```bash
echo "Hello, World!"
```
````

**表示例：**
```javascript
function hello() {
  console.log("Hello, World!");
}
```

## テーブル

GitHub Flavored Markdownのテーブル記法が使用できます。

```markdown
| 列名1   | 列名2   | 列名3   |
| ------- | ------- | ------- |
| データ1 | データ2 | データ3 |
| データ4 | データ5 | データ6 |
```

**表示例：**
| 列名1   | 列名2   | 列名3   |
| ------- | ------- | ------- |
| データ1 | データ2 | データ3 |
| データ4 | データ5 | データ6 |

### テーブルの配置

```markdown
| 左揃え | 中央揃え | 右揃え |
| :----- | :------: | -----: |
| 左     |   中央   |     右 |
```

## 使用できない記法

以下の記法は現在の実装ではサポートされていません：

- 画像（`![alt](url)`）
- チェックボックスリスト（`- [ ]`）
- 取り消し線（`~~text~~`）
- タスクリスト
- フッター/脚注

---

## 実装されている要素の一覧

現在の実装でスタイリングされている要素：

- ✅ 見出し（h2, h3, h4）
- ✅ 段落（p）
- ✅ 順序なしリスト（ul, li）
- ✅ 順序付きリスト（ol, li）
- ✅ テーブル（table, thead, th, td）
- ✅ インラインコード（code）
- ✅ コードブロック（pre, code）
- ✅ 引用（blockquote）
- ✅ リンク（a）
- ✅ 太字（strong）

## 使用例

実際のセクションファイルでの使用例：

```markdown
---
title: "セクションタイトル"
order: 1
---

# セクションタイトル

このセクションでは、以下の内容を学習します。

## 主要なポイント

1. **重要な概念**
   - ポイント1
   - ポイント2

2. **実践的な例**

| 項目     | 説明         | 例           |
| -------- | ------------ | ------------ |
| データ型 | データの種類 | 文字列、数値 |

### コード例

```javascript
const example = "Hello, World!";
```

> 注意：この例は参考用です。
```

---

## 補足

- セクションファイルのフロントマター（`---`で囲まれた部分）には`title`と`order`が必要です
- セクション内の見出しは`##`から始めることを推奨します（`#`はセクションタイトルとして使用されるため）
- テーブルはGitHub Flavored Markdown形式で記述できます
